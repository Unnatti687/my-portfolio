Who Am I?

I am a Computer Science & Engineering student deeply fascinated by the invisible architecture that powers the modern web. My journey bridges the gap between scalable cloud systems and intuitive frontend experiences.

Why Cloud Computing? Because I believe in building systems that last. I am actively working with AWS—specifically Amazon S3, IAM, and CloudFront—to understand how to deploy secure and globally accessible applications. I don't just want to write code; I want to architect environments where code thrives.

At the same time, I study Web Development to maintain empathy for the end-user. Mastering HTML, CSS, and JavaScript allows me to see the full picture—from the server rack to the browser screen. My goal is to become a Cloud & DevOps Engineer who builds with both performance and human experience in mind.
